# Project: Argos3
A Argos3 mulit-robot simulation project running on a Vagrant Ubuntu Xenial VM guest Windows host.

## Requirements:
* Vagrant 1.9.5
* Virtual Box 5.1.22
* Windows 8

## Instructions
1. Navigate to repo root directory 
2. Start machine:
```bash
vagrant up
```
3. Connect to machine:
```bash
vagrant ssh
```
4. Navigate to folder:
```bash
cd /vagrant
```
